# ios-tools
Code, scripts, actions, docs, tools for Java on iOS

## Github Actions

This repository contains scripts to build the components needed to run Java on iOS.
They are documented [here](docs/ga.md)

## Documentation

The repository also contains documentation for developers interested
in helping to maintain OpenJDK for iOS.
See [here](docs/jdkinternals.md) for a document-in-progress
explaining some of the relevant internals of the OpenJDK.
